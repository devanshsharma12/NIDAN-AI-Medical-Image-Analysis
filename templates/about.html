About This Project

This Medical Image Analysis System is designed to assist doctors and researchers by automating the diagnosis of critical health conditions through advanced deep learning models. The system intelligently classifies and segments medical images such as eye scans, brain MRIs, and chest X-rays.

Multi-Disease Detection

Supports classification for Diabetic Retinopathy, Brain Tumors, COVID-19, Pneumonia, and more using dedicated deep learning models.

Automated Segmentation

Identifies and highlights infected regions in chest X-rays and brain MRIs with pixel-wise accuracy using U-Net-based segmentation models.

Smart Image Classification

Automatically detects whether the input image is from the eye, brain, or chest, and routes it to the correct model pipeline.

Confidence Scoring

Each prediction comes with a confidence percentage, helping medical professionals understand the certainty of the result for better decision-making.

Grad-CAM Visualization

Highlights key regions in the image that influenced the model’s decision, providing transparency and trust in the AI system’s output.

Web-Based Interface

Easily accessible through a clean and responsive web app built with Flask and Tailwind CSS, allowing image uploads, results view, and downloads.

Multi-Model Support

Automatically selects between eye disease and brain tumor models using image type detection for accurate diagnosis and analysis.

Downloadable Results

Users can download segmented images with highlighted tumor regions, enabling easy sharing with medical professionals.

Interactive UI & Animations

Enhanced with Tailwind CSS and subtle animations for a modern, engaging user experience with dynamic effects like zoom and blink.

Project Motivation

Developed as a B.Tech final year project by a passionate team of AI enthusiasts, this system aims to bridge the gap between medical diagnostics and intelligent software solutions. It is a step toward accessible and scalable healthcare diagnostics.
