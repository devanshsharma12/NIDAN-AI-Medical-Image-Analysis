📄 NIDAN AI – Project Documentation

Welcome to the official documentation page of NIDAN AI, our final year B.Tech CSE project. This intelligent medical image analysis system helps detect, classify, and segment diseases in eye fundus images and brain MRIs using deep learning models.

📘 Project Overview

NIDAN AI is designed to assist healthcare professionals by providing quick and accurate disease detection through AI. It uses Flask as the backend, Tailwind CSS for styling, and integrates CNN, EfficientNet, and U-Net models for multi-task medical image processing.

  • Classifies eye diseases like Diabetic Retinopathy
  • Detects and segments brain tumors
  • Automatically identifies the type of uploaded image
  • Displays Grad-CAM visualizations and segmented masks
